# npm-build-tool-tutorial

A simple and helpful Build-Tool process with npm

## How to use

```
npm run test

npm run pretest

npm run posttest

npm run start

npm run start:dev

npm run lint

npm run precompile

npm run clean

npm run compile

npm run compile:ts

npm run "build:less

npm run build:browserify

npm run build:bundle

npm run build:clean

npm run prebuild

npm run build

npm run watch:bundle
```
